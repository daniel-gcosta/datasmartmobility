# Aula 3 - Dados em Mobilidade Inteligente: técnicas de processamento e visualização

Este curso está focado no _processamento e na visualização de dados urbanos_ que estão associados (direta e indiretamente) à mobilidade inteligente. Serão abordadas técnicas para coleta, tratamento e interpretação de pequenos e grandes volumes de dados provenientes de diferentes fontes. O objetivo é apoiar a _tomada de decisão e o planeamento urbano_ por meio do "uso" eficiente de dados.

É importante lembrar que este curso é centrado na escrita de código (programação) e está organizado em diferentes partes. No geral, não é preciso possuir conhecimentos avançados em nenhuma linguagem de programação ou recurso computacional. O curso foi concebido para atender a um **público heterogêneo**, composto por estudantes e profissionais de diferentes áreas, com variados níveis de familiaridade com tecnologia e programação. Portanto, não é esperado que os participantes mergulhem profundamente nos detalhes técnicos da programação, mas sim que desenvolvam uma compreensão geral dos processos envolvidos e das possibilidades oferecidas pelo uso de dados no contexto urbano.

Nesse sentido, o foco está em compreender **quais tipos** de dados existem, **como** esses dados são captados, processados e analisados, e **quais ferramentas e métodos** são utilizados para apoiar a tomada de decisão em projetos de mobilidade inteligente. Essa visão ampla permite que os participantes dialoguem com equipas técnicas, interpretem resultados analíticos e incorporem o uso de dados de forma estratégica no planeamento e na gestão da mobilidade urbana.

**Parte 1 - Visualização de dados urbanos com OpenStreetMap**

Ao longo desta parte, será utilizado o OpenStreetMap como uma importante fonte de dados geoespaciais. Serão estudadas diferentes formas de obtenção, exploração e uso desses dados, com o objetivo de identificar padrões de mobilidade, compreender a estrutura do sistema viário e reconhecer desafios urbanos relacionados ao deslocamento de pessoas e veículos. O uso de dados abertos permitirá analisar o território de forma integrada, apoiando diagnósticos e decisões em projetos de mobilidade inteligente. O foco será, portanto, na **visualização de dados**.

**Parte 2 - Manipualção de datasets de mobilidade**

A análise de dados será abordada a partir da perspectiva de Data Science, combinando diversos métodos para extrair conhecimento a partir de bases de dados de mobilidade. O curso fará uso de dados provenientes do Eurostat, permitindo contextualizar fenômenos de mobilidade em escalas regional e europeia, bem como comparar indicadores entre diferentes territórios. Além disso, serão introduzidos conceitos e aplicações de Inteligência Artificial, com foco no uso de algoritmos e modelos para apoiar a análise, a interpretação de padrões e a geração de insights a partir de grandes volumes de dados de mobilidade.
