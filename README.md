# Dados em Mobilidade Inteligente: técnicas de processamento e visualização

Este curso está focado no processamento e na visualização de dados urbanos que estão associados à mobilidade inteligente. Serão abordadas técnicas para coleta, tratamento e interpretação de pequenos e grandes volumes de dados provenientes de diferentes fontes. O objetivo é apoiar a tomada de decisão e o planejamento urbano por meio do uso eficiente de dados.

Este curso é centrado na escrita de código (programação) e está organizado em diferentes partes. No geral, não é preciso conhecimentos avançados em nenhuma linguagem de programação ou recurso.
